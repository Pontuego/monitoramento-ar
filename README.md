# Monitoramento de Ar

Sistema de monitoramento de qualidade do ar com coleta de dados em tempo real, visualização e alertas automatizados.

## 📖 Sobre o Projeto

O **Monitoramento de Ar** é uma aplicação desenvolvida para acompanhar métricas ambientais e auxiliar no monitoramento da qualidade do ar em diferentes ambientes. O projeto permite coletar dados de sensores, armazenar informações e visualizar métricas importantes de forma organizada.

Esse tipo de solução é amplamente utilizado em projetos de IoT, automação e observabilidade ambiental. ([Reddit][1])

---

## 🚀 Funcionalidades

* 📡 Coleta de dados em tempo real
* 🌡️ Monitoramento de temperatura e qualidade do ar
* 📊 Dashboard para visualização dos dados
* ⚠️ Sistema de alertas para valores críticos
* 🗄️ Armazenamento de histórico das medições
* 🔄 Atualização automática das informações

## 🛠️ Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando tecnologias modernas para monitoramento e visualização de dados:

## ⚙️ Como Executar o Projeto

### Pré-requisitos

Antes de começar, você vai precisar ter instalado:

* [Node.js](https://nodejs.org?utm_source=chatgpt.com)
* [Git](https://git-scm.com?utm_source=chatgpt.com)

---

### Clone o repositório

```bash
git clone https://github.com/Pontuego/monitoramento-ar.git
```

---

### Acesse a pasta do projeto

```bash
cd monitoramento-ar
```

---

### Instale as dependências

```bash
npm install
```

---

### Execute o projeto

```bash
npm run dev
```
